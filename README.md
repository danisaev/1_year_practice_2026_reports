# Отчёты по учебным практикам 1 курс (второй семестр) 2026 года. Весна 2026.
Как добавить свой отчёт

1. Сделать форк репозитория.
2. Создать директорию с названием `<Nmae_Surname>` (да, на английском). Как в примере.
3. Поместить в директорию `.tex`-файл с отчётом (ровно по тому шаблону, который вам предлагалию В целом, его можно взять из примеров же). Назвать `<Name_Surname>_report.tex`. Как в примере. Если нужны вспомогательные ресурсы (например, изображения), то разместить их рядом, в этой же директории (можно и нужно использовать поддиректории для структурирования). Если используете изображения в формате pdf (что, в целом, правильно), то их, возможно, придётся добавлять как `git add --force`, так как всё, что с расширением `.pdf` сейчас игнорируется, чтобы сами отчёты скомпилированные не выкладывали.
4. Проверить, что локально собирается. В своей поддиректории запустить
   ```console
   $ latexmk -lualatex -interaction=nonstopmode -file-line-error -shell-escape <Name_Surname>_report.tex
   ```
   Убедиться, что не только сгенерировалась pdf, но и не было ошибок компиляции. Компиляторы TeX-а любят даже при наличии ошибок сгенерировать "хоть что-нибудь".
5. Добавить строчку вида `\includepdf[pages=-]{<Nmae_Surname>/<Nmae_Surname>_report.pdf}` в `main.tex` по аналогии с уже добавленными.
6. Открыть реквест со всеми этими изменениями в данный репозиторий в ветку `main`. Убедиться, что CI прошёл.
